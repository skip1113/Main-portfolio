Trails on Tap

Trails on Tap is a serach and plan application using Hiking Project, Open Brewery, and Geocoding API. If you like to go on hikes, and drink craft beers this is the app for you. This allows the user to be able to plan a hike and grab a drink with only one search of either State, City, or Zip code resulting in trails, and breweries within the radius of your search.
One problem was that we had to overcome searching by cooardinates and simplify to serach by City or Zip. We used The Geocoding API to convert the search of City or Zip to Longitude and Latitude so the hiking API would be able to find trails.
